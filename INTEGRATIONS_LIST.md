## üöÄ Adding New Integrations

To add a new integration:

1. **Add API key to `.env.example`**
   ```env
   # New Provider
   NEW_PROVIDER_API_KEY="..."
   ```

2. **Add to config** (`backend/app/core/config.py`)
   ```python
   NEW_PROVIDER_API_KEY: str = Field(default="")
   ```

3. **Create service** (`backend/app/services/new_provider.py`)
   ```python
   class NewProviderService:
       async def call_api(self, ...):
           # Implementation
   ```

4. **Add to fallback logic** (if applicable)
   ```python
   providers = [user_key, project_default, free_tier]
   ```

5. **Update documentation** (add to this list)
